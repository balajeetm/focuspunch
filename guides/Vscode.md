## Shortcuts

| Item                | Command            | Description |
| :------------------ | :----------------- | :---------- |
| Single line comment | Cmd + /            |             |
| Block comment       | Shift + Option + A |             |
| Extract variable    | Cmd + .            |             |
| Delete Line         | Cmd + Shift + K    |             |
| Move backward       | Ctrl + -           |             |
| Move forward        | Ctrl + Shift + -   |             |
